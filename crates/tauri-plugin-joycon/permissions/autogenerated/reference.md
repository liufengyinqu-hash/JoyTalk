## Default Permission

Default permissions for the Joy-Con plugin

#### This default permission set includes the following:

- `allow-get-status`
- `allow-get-mappings`
- `allow-set-mapping`
- `allow-reset-mappings`
- `allow-set-enabled`
- `allow-get-enabled`
- `allow-list-actions`
- `allow-start-capture`
- `allow-stop-capture`
- `allow-list-presets`
- `allow-get-preset-mappings`
- `allow-load-preset`
- `allow-load-preset-from-url`
- `allow-export-mappings`
- `allow-import-mappings`
- `allow-list-apps`
- `allow-get-imu`
- `allow-set-imu`
- `allow-get-ir`
- `allow-set-ir`
- `allow-get-ir-sample`
- `allow-get-nfc-sample`
- `allow-restart-nfc-scan`
- `allow-get-profiles`
- `allow-save-profile`
- `allow-delete-profile`
- `allow-set-per-app-enabled`
- `allow-get-per-app-enabled`
- `allow-get-frontmost`
- `allow-get-frontmost-app`
